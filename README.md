# Kimi K2.6 UDE

**Unified Developer Environment** — A mobile-first code editor with AI-powered assistance, built with Capacitor for Android.

## Features

- 🧠 **Kimi K2.6 AI** — Powered by NVIDIA NIM, integrated chat assistant
- 💻 **Code Editor** — Syntax-aware editor with line numbers and auto-indent
- 🖥️ **Terminal** — Built-in terminal with file management commands
- 📁 **File Manager** — Create, edit, import, and delete files
- 🎨 **Dark/Light Themes** — Beautiful Theia-inspired dark theme
- ⌨️ **Keyboard Shortcuts** — Ctrl+S save, Ctrl+Enter ask AI, Ctrl+` terminal
- 📱 **Android APK** — Built with Capacitor 6, supports ARM64 + ARMv7

## Building

The APK is built via GitHub Actions. Push to `main` or trigger manually.

## API Configuration

1. Open the app
2. Tap ⚙️ Settings
3. Enter your NVIDIA NIM API key (`nvapi-...`)
4. Select model: `moonshotai/kimi-k2.6`
5. Start chatting with AI!
